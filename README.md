# Laravel Deployment & CI/CD Skill

An AI agent skill for production deployment pipelines, GitHub Actions CI, Docker, and zero-downtime deploys.

## Topics Covered

- GitHub Actions CI workflow (PHP, MySQL, Redis, Pint, PHPStan, Pest)
- Zero-downtime deployment with symlink switching
- Docker setup (Dockerfile, docker-compose, nginx, supervisor)
- Production checklist (security, monitoring, infrastructure)

## Installation

```bash
npx skills add neuraliq/laravel-deployment-cicd
# or
php artisan boost:add-skill neuraliq/laravel-deployment-cicd
```

## Compatibility

Claude Code, Cursor, Windsurf, GitHub Copilot

## Requirements

Laravel 11+, GitHub Actions, Docker (optional)
